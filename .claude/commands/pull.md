현재 작업 디렉토리에서 드라이브 문자를 자동 감지하여 `{드라이브}:/Folder/claude/` 하위의 모든 git 저장소에서 git pull을 실행하라.

## 드라이브 감지
- `pwd` 또는 현재 작업 디렉토리 경로에서 드라이브 문자를 추출 (예: C, D)
- 감지된 드라이브로 `{DRIVE}:/Folder/claude/` 경로를 결정

## 대상 저장소
- `{DRIVE}:/Folder/claude/*/` 중 `.git` 디렉토리가 있는 모든 저장소를 자동 발견

## 실행 방법
1. `{DRIVE}:/Folder/claude/*/` 를 순회하며 `.git`이 있는 디렉토리를 찾는다
2. 각 저장소 디렉토리로 이동하여 `git pull` 실행
3. 결과를 저장소별로 간단히 요약 (업데이트 유무, 변경된 파일 수)
4. 모든 저장소 pull 완료 후 최종 요약 출력
